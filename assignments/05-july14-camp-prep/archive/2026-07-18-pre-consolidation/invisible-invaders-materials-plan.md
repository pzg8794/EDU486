## Planning Assumptions

- We are planning for **four shared stations**, approximately **12-16 campers in groups of 3-4**. The final camper count has not been provided, so quantities scale by station instead of assigning one of everything to every camper.
- Each station must support an indoor sample, an outdoor sample, and a blank. Matching tools and the same record fields make the comparison more trustworthy.
- Campers can observe by eyepiece, shared camera view, still image, or partner description. They can record by paper, typing, dictation, photograph, drawing, pointing, or sorting cards.
- Color can make stations easier to track, but **every color code also receives a written sample ID and symbol**.
- Existing equipment is not considered ready until it is counted, cleaned, tested, and reserved.
- Visually observed particles are recorded as **suspected microplastics** unless an appropriate polymer-identification method confirms them.

## Order Priority If Prices Rise

1. Keep filter paper, slides, clear tape, and foil first because they protect the sample-to-evidence chain.
2. Keep both glove sizes unless the physical check confirms appropriate powder-free stock in several sizes.
3. Keep permanent markers because every sample needs a durable written ID.
4. Remove colored masking tape first if the total exceeds $100; use existing labeling tape with written IDs and symbols.
5. Remove glue sticks next only if the LeChase low-odor adhesive stock is confirmed usable.
