## Order Decision Rule

1. Name the activity and evidence it must produce.
2. Check whether an existing material can serve the same purpose safely.
3. Identify quantity, reusable status, delivery deadline, and team.
4. Check sensory, allergy, mobility, dexterity, cleanup, and low-energy participation needs.
5. Add only the confirmed need to the [camp materials order sheet](https://docs.google.com/spreadsheets/d/1rAuGt8iT-CAx4Oy1cqz_SsVQqqPzi5mE9v7pRCODSAo/edit?gid=1535708448#gid=1535708448).

Each team has a $100 budget. The instructors said to ask before assuming a larger need cannot be supported, and warned that Amazon Prime timing is not fully reliable.
